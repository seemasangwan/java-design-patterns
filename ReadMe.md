# 🚀 Java Design Patterns

This repository contains implementations of **GoF Design Patterns** in Java with real-world examples.

---

## 📂 Folder Structure

- **Creational Patterns:** Factory, Abstract Factory, Builder, Singleton  
- **Structural Patterns:** Adapter, Decorator, Proxy  
- **Behavioral Patterns:** Strategy, Observer, Command  

---

## 🏗️ Examples

### Factory Pattern
- Example: Car Factory producing different types of cars  

### Singleton Pattern
- Example: Logger class, DB connection pool
